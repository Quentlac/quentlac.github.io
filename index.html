MES PROJETS

APPLICATION RÉSERVATION EN LIGNE DE STAGE DE VOILE

Développement d'une application web pour le club de voile YCBL permettant aux clients du club de réserver un stage pour l'été en ligne. L'application intègre une partie réservation dédié aux clients ainsi qu'une partie administration pour les employés du club afin de leur permettre de gérer les commandes et les offres proposées.

Pour le développement de cette application, j'ai tout d'abord étudiés les besoins nécessaires afin de rendre l'expérience utilisateur la plus agréable possible. D'un point de vue technique, il s'agit d'une application client / serveur avec une base de données PostgreSQL. Les technologies utilisées sont React pour le front-end, NodeJS pour le back-end.

J'ai également mis en place un système de paiement en ligne pour permettre aux clients de payer leur réservation en ligne. J'ai pour cela utilisé l'API ETransaction proposé par la banque Crédit Agricole.

L'application possède un système d'envoi de mails automatiques et de génération de facture afin de les envoyer aux clients après leur réservation.

Cette application intègre un système de stock et permet de moduler le nombre de places disponibles pour chaque stage et pour chaque semaines. Il est possible de créer des packages pour ajouter des options aux stages. Cela faisait partis des attentes du club de voile pour lequel j'ai travaillé.

Le site est aujourd'hui activement utilisé par le club de voile afin de remplir ses stages pour la saison estivale. Vous pouvez accéder au site en cliquant sur le bouton ci-dessus.

PROJET GAME JAM : GRAVITY ESCAPE

Réalisation d'une Game Jam en Python organisé par l'IUT2 de Grenoble au sein de la promo de 3ème année de BUT Informatique. L'objectif était de créer un jeu vidéo en Python avec la bibliothèque pygame en 1 semaine par équipe de 4 personnes. Le thème imposé était Maître de la gravité, l'histoire d'un robot.

Nous avons donc réalisé le jeu que nous avons nommé "Gravity Escape". Il s'agit d'un jeu vidéo de type platforme où le joueur incarne un robot qui doit réussir différents niveau. Pour y parvenir il dispose d'un canon qui lui permet de tirer des projectiles pour modifier la gravité de certains blocs.

J'étais chargé du développement de la physique du jeu. Je me suis donc occupé de créer le moteur physique en utilisant une approche orienté objet afin de faciliter le développement.

Afin de travailler à plusieurs sur le projet, j'ai mis en place un dépôt Github permettant de collaborer sur le projet

Ce projet nous a valu la seconde place à la Gamejam 2023 organisé par l'IUT.

RÉSEAU SOCIAL : CHEERLAND

Dans le cadre d'un projet personnel, j'ai réalisé une application mobile de type réseau social nommé "Cheerland". L'objectif était de créer une application permettant de partager du contenu de manière innovante et originale. Le principe consiste en une map infinie sur laquelle les utilisateurs peuvent se déplacer un peu comme sur Google Maps. Cette "map" est de base vierge et n'est qu'une grille vide. Les utilisateurs peuvent y ajouter des éléments comme des messages ou même des photos. Les autres utilisateurs peuvent alors voir ces éléments en se déplaçant sur la map et y réagir directement.

L'application mobile était conçu à la fois pour Android et iOS grâce à la technologie React Native. Pour le côté backend, j'ai utilisé PHP pour créer une API avec laquelle l'application communique pour récupérer les données à afficher sur la map.

J'ai également mis en place un système de notifications push pour informer les utilisateurs des réponses d'autres utilisateurs à leur posts (nommé cheers). Pour cela, j'ai utlisé le framework expo pour React Native qui permet de gérer les notifications push.
